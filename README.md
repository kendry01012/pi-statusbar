## 📝 What is pi-statusbar?

pi-statusbar is a simple tool that shows important information for pi.dev agents right on your Windows taskbar. It helps you keep track of your agent’s status and activity without opening any extra windows. This tool works quietly in the background and updates info in real time.  

You don’t need any technical skills to use pi-statusbar. Just install it on your Windows system, and it will run smoothly with minimal setup.

---

## 💻 System Requirements

Before installing pi-statusbar, make sure your computer meets these requirements:

- Windows 10 or later  
- At least 1 GB of free disk space  
- An active internet connection for updates and data syncing  
- A screen resolution of 1024x768 or higher

After installation, you will see a small icon for pi-statusbar near the bottom right corner of your screen, in the Windows taskbar notification area.

## 🛠️ Basic Configuration

You may want to adjust pi-statusbar settings when you first start it. Here’s how:

1. **Open Settings**  
   Right-click the pi-statusbar icon in the notification area and choose “Settings.”

2. **Adjust Startup Options**  
   You can choose if you want pi-statusbar to start automatically when you turn on your computer.

3. **Change Display Preferences**  
   Select what information the statusbar shows, like agent availability, latest tasks, or alerts.

4. **Set Update Frequency**  
   Decide how often the statusbar refreshes data. The default is every 30 seconds.

5. **Save and Exit**  
   Click “Apply” or “OK” to save your changes. The new settings take effect immediately.
